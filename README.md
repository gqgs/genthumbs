CLI to create video mosaics.

### Installation

If you're on Linux a mpv user script can be create with:

```
make mpv
```

### Usage

The default keybind to generate previews is the 'g' key.

### Dependencies
 - `ffmpeg`
 - `ffprobe`
 - `xxh128sum (optional)`
